# 카메라 룩 라이브러리 (prefix 후보)

영화의 **prefix**(공통, 영화 단위 상수)를 정하거나 톤을 바꿀 때 쓰는 메뉴. 카메라 종류/렌즈/필름
스톡/그레이드는 **영화 전체가 통일**돼야 하므로 prefix에 고정한다(샷마다 반복 X). 고유명사는 영어가
토큰 인식이 잘 돼서 영어 유지.

## 테스트 원칙

씬은 고정하고 **prefix(룩) 한 블록만 바꿔서** A/B 비교한다. 안 그러면 무엇이 효과인지 분리 안 됨.

## 실전 메모 (검증됨)

- **샷별 컨트롤(사이즈·카메라 무브·연기)이 가장 확실한 레버다.** 명시할수록 결과가 또렷이 따라온다.
- **필름 스톡(Kodak/CineStill/16mm 등) 차이는 엔진에서 미묘**할 수 있다 — prefix는 가볍게 한 톤만
  잡고, 룩을 필름 스톡 A/B로 과하게 파지 말 것. 차이가 안 느껴지면 굳이 스톡 용어를 늘리지 않는다.

## 빌딩블록

### 렌즈 / 화각 — 압축감·친밀감
- `85mm 망원 렌즈, 얕은 심도, 부드러운 보케` — 인물 압축, 배경 분리(클로즈업)
- `24mm 광각 렌즈` — 공간감·왜곡, 역동적
- `anamorphic lens, 가로 렌즈 플레어, 타원형 보케` — 시네마스코프 룩(프레임 비율은 출력 포맷에서)
- `macro lens, 극단적 클로즈업` — 인서트(눈·손·소품)

### 필름 스톡 / 에뮬레이션 — 색·그레인
- `shot on Kodak Vision3 500T` — 따뜻한 영화적 컬러
- `CineStill 800T, halation, 네온 글로우` — 야간 네온 번짐(누아르)
- `16mm film grain, 빈티지` — 거친 아날로그
- `65mm large format, 초선명` — 대형 포맷 디테일

### 셔터 / 모션
- `180도 셔터, 자연스러운 모션 블러` — 표준 영화 모션
- `고속 셔터, 또렷하고 스타카토한 움직임` — 긴박감
- `슬로우 모션 120fps` — 감정 강조

### 카메라 무브 / 그레이드
- 무브: `Steadicam 부드러운 트래킹` / `핸드헬드 다큐멘터리` / `크레인 상승샷` / `dolly zoom(현기증)`
- 그레이드: `teal & orange` / `bleach bypass, 탈색` / `low-key 키아로스쿠로` / `골든아워 림라이트`

> 주의: **카메라 무브**는 prefix가 아니라 **샷별 본문**에 쓰는 게 맞다(샷마다 달라짐). 여기 무브
> 항목은 영화 전체의 기본 무빙 성향을 prefix에 깔고 싶을 때만 참고.

## 바로 쓰는 룩 조합 (prefix 통째 교체용)

**A. 클래식 시네마틱**
```
shot on ARRI Alexa, 35mm 렌즈, Kodak Vision3 필름룩, 180도 셔터, teal & orange 그레이드, 얕은 심도
```

**B. 아나모픽 블록버스터** (화면비는 출력 포맷에서 2.39:1 / 21:9 설정 — 프롬프트에 비율 토큰 넣지 말 것)
```
anamorphic lens, 40mm, 가로 렌즈 플레어, 타원형 보케, shot on ARRI, 얕은 심도
```

**C. 빈티지 16mm**
```
16mm film grain, 빈티지 렌즈, halation, 탈색된 색감, 핸드헬드
```

**D. 야간 네온 누아르**
```
CineStill 800T, 네온 실사광, low-key 조명, halation, shallow depth of field
```

확정된 룩은 `data/prompt-preset.yaml`의 `prefix`에 저장한다.

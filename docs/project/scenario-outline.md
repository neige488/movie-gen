# PTSD (가제) — 씬 인벤토리 v2

> **이 문서의 사용법**
> - `concept.md` v2(규칙)와 짝을 이루는 **씬 전수 목록.** 가능한 씬을 모두 적고, 편집에서 고른다 — "테스트 후 판단."
> - 시간 모델(concept 1장)에 따라 **씬 배치 순서는 잠정**이다. 환상 씬들 사이 시간축 정합은 요구하지 않는다. 막 구분은 위장플롯(1막 가까워짐 / 2막 멀어짐 / 3막 같이 집 찾기) 기준의 잠정 배치.
> - 각 씬 표기:
>   - **축:** `공통`(거울 축 무관) / `A`(횡단보도 회귀 전용) / `B`(스위스 전용)
>   - **층위:** `현실` / `환상` / `회상-함정` / `모호`(감독은 답을 정해둘 것)
>   - **상태:** `✅ 대사 확정` / `프롬프트 있음` / `미작성`
>   - [위장] 첫 관람이 읽는 것 / [진실] 재관람에 드러나는 것
> - 의상 규칙 상기: 환상 수아 = 사고 당시 옷 한 세트 고정(빨간 운동화 포함). 교복 = 현실 표지.

## 현실 원장 ↔ 씬 매핑 (감독용 닻)

| 현실 원장 (concept 1-1) | 대응 씬 |
|---|---|
| 1. 교실 창문 바라봄 + 눈맞춤 | C-02 |
| 2. 횡단보도 사고 (수아의 죽음) | C-20 (파편), C-01과 공통 구간 공유 |
| 3. 책이 지훈의 손에 남음 | C-12 (전치된 형태로) |
| 4. 바람을 혼자 실행 | C-09R, C-10R (혼자 버전들) |
| 5. 무너짐 (공황, 병원) | C-17, C-18, C-21 |
| 6. 책 반납 (수아의 집, 엄마) | C-22, C-23 |
| 7. 이후의 삶 | END-B (3년 뒤) |

나머지 모든 씬은 덧붙여진 기억(환상)이거나 함정이다.

## 오프닝 후보 [OPEN — 거울 축과 연동]

### OP-A. 시드 1b — 횡단보도 (A축 오프닝)
- **축:** A · **층위:** 모호(사고 전조와 환상의 접면) · **상태:** 프롬프트 있음 (테스트 자산)
- 횡단보도 질주 → 1인칭 스쳐지나감 → 슬로우모션 3/4 얼굴, 빨간불.
- A축 채택 시 오프닝이자 거울 축의 한쪽. B축 채택 시 중반(사고 씬 인접)으로 이동 가능.

### OP-B. 스위스 로드트립 — 익명 처리 (B축 오프닝)
- **축:** B · **층위:** 모호(엔딩에서야 닫힘) · **상태:** 미작성
- 차 안, 설산을 보며 주행 → 디졸브 → 제목. **익명 절충 [채택 기울어짐]:** 운전자 안 보임, 손·창밖 설산·도로만, 한국어 단서 없이. 첫 관람 = 익명의 길 / 엔딩에서 "그 길이 여기로 오는 길"로 닫힘.
- ⚠️ 위험(기록): 오프닝 스위스가 엔딩 도착 충격을 미리 죽일 수 있음 / "한국 멜로" 위장 약화. 익명 처리가 완화책.

### OP-C. 공통 구간 — 집·골목 질주 [공통 클립, 재호출용]
- **축:** 공통 · **층위:** 모호(현실의 그날이자 환상의 입구) · **상태:** 미작성
- INT. 지훈의 집 — 부산한 학원 준비(게임하다 늦음) → EXT. 골목 질주, 거친 숨 → EXT. 횡단보도 도착, 빨간불.
- **[제작]** 같은 소스를 오프닝/사고/엔딩-A 재호출에 공용. 재호출 디테일 심기: 후드티 끈, 반쯤 닫힌 가방 지퍼.

## 1막 — 연인 만들기 (덧붙여진 기억)

> **[확정] 데이트 씬은 전부 1막.** 1막의 임무는 둘을 최대한 완전한 연인으로 만드는 것. 환상 씬의 정보는 전부 출처 짝(지식 장부)이 있어야 한다: 책 메모·교실 관찰·환경 파편. 톤: 너무 사실적이면 안 됨 — *어딘가 너무 완벽해 비현실적인* 결을 얇게. 말투: 40대가 추억하는 밝고 귀여운 첫사랑(concept 11장).

### C-01. 빨간불 문답 — 첫 만남이자 영원한 재생 ✅
- **축:** 공통 · **층위:** 환상 · **상태:** ✅ 대사 확정 (테스트 완료)
- 수아가 고개를 들며: "너 또 빨간불에 건너려 했지?" / "아니거든." / "거짓말." / "넌 어떻게 맨날 알아?" / **"네가 맨날 그러니까."** → 초록불, 같이 건넘.
- [위장] 친한 둘의 풋풋한 일상. [진실] "네가 맨날 그러니까" = ① 빨간불에 건너는 습관(사고 원인)의 복선 ② 이 장면을 *맨날* 머릿속에서 반복 구성한다는 루프의 자백. ③ 수아가 그를 **멈춰 세운다** = 환상의 분기점("멈춰 서서 말을 건넸다면")의 성취 — 현실에선 아무도 그를 멈춰 세우지 않았다.
- **위치 자유 [확정]:** 시간 모델 덕에 "이미 친함" 전제와 1막 "가까워짐"이 충돌하지 않음. 오프닝/1막 중반 어디든 배치 가능.

### C-02. 교실 창문 — 사전 접촉 (신규) ★함정 장치
- **축:** 공통 · **층위:** 현실(회상) — 눈맞춤 1회의 현실 여부만 [감독 보유] · **상태:** 미작성
- 지훈이 수아의 교실 앞을 지나다 창문 너머로 책에 집중한 수아를 슬쩍 봄. 몇 번 반복. 어느 날 수아가 고개를 들어 지훈을 바라봄 (한 번).
- **수아는 교복** (현실 표지).
- [위장] 풋풋한 짝사랑의 시작. [진실] 이것이 둘의 관계의 *전부*였다. 그리고 사고 직후 "아는 얼굴이라 돌아봄"(C-20 연출 비트 ②)과 환상의 분기점("멈춰 서서 말을 건넸다면")의 출처.
- **★함정 용법:** 이 회상을 지훈이 수아를 부르는 씬들 사이에 삽입 → 관객은 "수아가 교복도 입는구나"로 흡수. 재관람: 교복 장면만이 현실. (#충돌의 교복 여학생 함정과 한 세트.)

### C-03. 편의점 — "천천히 먹으면 오래가"
- **축:** 공통 · **층위:** 환상 · **상태:** 대사 초안 (데모) — 출처 짝 필요
- 지훈: "그거 하나로 저녁이 돼?" / 수아: "천천히 먹으면 오래가." / "그게 무슨 말이야." / "빨리 먹으면 없어지잖아."
- [위장] 귀여운 식성. [진실] 가난을 매력으로 번역한 핵심 파편 — 실제론 먹을 게 없었음.
- ⚠️ 지식 장부: 이 정보의 출처(메모/관찰 파편)를 인벤토리상 어느 씬이 제공하는지 확정 필요 [OPEN].

### C-04. 비 오는 골목 — 우산
- **축:** 공통 · **층위:** 환상 · **상태:** 대사 초안 (데모)
- "너보다 책이 더 중요하냐?" / "책은 젖으면 티 나잖아." / "사람은?" / **"사람은 말리면 돼."**
- [위장] 책벌레의 엉뚱함. [진실] 자기를 돌보지 않는 아이. 그리고 젖지 않고 남은 것은 결국 책뿐이었다.

### C-05. 버스정류장 — "집에 가는 이야기"
- **축:** 공통 · **층위:** 환상 · **상태:** 대사 초안 (데모) — 메모 텍스트(미결 5)와 연동
- "맨날 뭘 그렇게 읽냐?" / "집에 가는 이야기." / "그게 재밌어?" / "집에 가는 게 제일 어려운 사람도 있잖아."
- [진실] 외로움/귀가 모티프 — 시집 밑줄 문장과 연결될 후보. 3막 "같이 집을 찾는다"와 수미.

### C-06. 무릎베개 거울 A — 벤치에서 책 읽기
- **축:** 공통 · **층위:** 환상 · **상태:** 프롬프트 있음 (테스트 자산)
- 언덕 벤치+우체통, 측면 투샷. 수아가 지훈 무릎에 누워 책 읽음. 따뜻한 빛.
- [제작] 거울 B(C-16)와 같은 벤치·같은 측면 구도 필수.

### C-07. 우체통 — 편지 넣는 수아
- **축:** 공통 · **층위:** 환상 · **상태:** 프롬프트 있음 (측면 투샷, POV 버전은 폐기됨)
- "여기 아무도 안 오는 거 알지? 근데 편지 넣어두잖아, 그럼 한 번씩 누가 싹 가져가. 나 [울적할/꿀꿀할] 때마다 여기 와서 편지 써서 넣어. 그럼 꼭… 누가 들어주는 거 같단 말이야." (말씨는 톤 가이드로 — 40대가 사랑할 밝고 귀여운 쪽.)
- [위장] 사랑스러운 습관. [진실] 응답 없는 곳에 계속 말 거는 것 — 지훈이 죽은 그녀에게 하는 짓의 거울.

### C-08. 스위스 대화 — "같이 가자"
- **축:** 공통 (엔딩만 B 전용; 바람 ③ 자체는 공통) · **층위:** 환상 · **상태:** 프롬프트 있음 (테스트 자산)
- 무릎베개, 책을 덮고: "호수+설산… 같이 가자."
- [진실] 시집 메모 ③(인터라켄)의 환상 실행 선언. B축 엔딩의 씨앗. ⚠️ "인터라켄"이라는 단어를 메모로 심는 타이밍이 사활 — 이르면 새고 없으면 안 터짐 [OPEN, 제목과 연동].

### C-09. 놀이공원 — 바람 ① 실행 쌍
- **축:** 공통 · **상태:** 미작성
- **C-09R (현실, 혼자):** 지훈이 혼자 놀이공원. 메모를 보고 온 것.
- **C-09F (환상, 둘):** 같은 장소·같은 블로킹에 수아 합성. 맑은 날, 웃음.
- [제작] 같은 씬 ref 교체 재생성 — 엔딩 "투샷→혼자 나열"(END-B')의 재료.

### C-10. 아이스링크 — 바람 ② 실행 쌍
- **축:** 공통 · **상태:** 프롬프트 있음 (환상 버전, 테스트 자산)
- **C-10R (현실, 혼자):** 혼자 능숙하게 탄다 (아빠에게 배웠다).
- **C-10F (환상, 둘):** 수아가 가르치고 지훈은 못 타는 척. **0.5초 능숙함 플리커** — 현실의 누수. 강조 금지.

### C-11. 불면 샷
- **축:** 공통 · **층위:** 현실 · **상태:** 미작성
- 침대의 지훈, 잠들지 못함. 1막에 무심히 배치.
- [위장] 설렘/일상. [진실] PTSD 불면 — 재관람 재맥락화.

## 2막 — 멀어짐 (붕괴)

### C-12. 가벼운 사고 목격 + 책 입수
- **축:** 공통 · **층위:** 모호 [감독 보유 — 정해둘 것] · **상태:** 미작성
- 지훈이 **혼자** 가벼운 접촉사고 목격. 여자아이는 멀쩡히 일어나 도망(**절대 죽지 않음**). 책이 그의 손에 남음.
- [위장] 우연한 목격, 책임감 있는 소년. [진실] 출처 전치 — 책의 진짜 출처(수아의 죽음)를 견딜 만한 가짜 출처로 바꿔치기.
- 🔲 사고 직후 **첫 번째 미세한 이상함** 미결 (잠깐 멍함 / 책을 못 버림 등) — 점증의 출발점.

### C-13. 뒷모습 집착
- **축:** 공통 · **층위:** 환상(위장상 "책 주인") · **상태:** 미작성
- 책 주인이 교복 입고 길에 앉은 **뒷모습으로만** 등장. 지훈이 흔적을 좇기 시작.
- [위장] 삼각관계 — 옆의 수아를 두고 정체 모를 여자에게 빠짐. [진실] 옆의 수아(환상) = 길의 뒷모습(죽은 수아) = 같은 사람. "자기가 자기를 질투"의 시각적 성립.
- **[LOCK]** 뒷모습은 끝까지 뒷모습/옆모습 — 지훈은 그녀 얼굴을 모른다. (7-2 폐기와 별개 — 이건 "책 주인" 페르소나의 규칙.)

### C-14. 빈자리 사다리 (3씬)
- **축:** 공통 · **층위:** 환상의 균열 · **상태:** 미작성
- ① 곁의 빈자리 — 반복 구도에서 수아만 빠짐 ② 분식집/친구 — 세계가 지훈에게만 응대 ③ **빈 책상 — 공인된 부재, 정점. 아껴서 늦게.**
- ⚠️ 확신이 굳기 직전 끊고 C-19(놀이터 재등장)로.

### C-15. 질투 / 싸움 — 균열의 정점
- **축:** 공통 · **층위:** 환상 · **상태:** 미작성 · 🔲 방아쇠 미결 (메모 텍스트 확정 후)
- 딱 한 번. "그 책 여자가 마음에 드는 거야?" — [위장] 연인 위기 / [진실] 전부 지훈이 자기에게 던지는 말. 카메라는 '지훈의 자문'으로.
- 이후 수아 거의 퇴장 (집에 갈 때만 재등장).

### C-16. 무릎베개 거울 B — 역전
- **축:** 공통 · **층위:** 환상 · **상태:** 프롬프트 있음 (테스트 자산)
- 같은 벤치, 같은 측면 구도. **지훈이 수아 무릎에 누워 욺.** 수아는 무표정(calm, unreadable)으로 쓰다듬음. 흐린 하늘, 식은 빛.

### C-17. #충돌 — 교복 여학생 ✅ ★함정 장치
- **축:** 공통 · **층위:** **현실** (보이는 수아 = 함정) · **상태:** ✅ 대사 확정, 프롬프트 있음
- 책상에서 자다 깬 지훈, 책이 없어진 것 발견 → 단발 여학생에게 달려가 따짐: "너지. 어디 숨겼어." / "말을 해." / "꺼져. 다시는 내 앞에 나타나지 마."
- **이름 금지 (concept 9장):** 상대는 수아가 아니라 자기 반의 다른 여학생. 지훈은 그 애에게 수아한테 말하듯 따진다. 책은 해리가 숨김. 보이는 수아 = 치마 교복 (C-02 회상 삽입이 만든 착각과 한 세트).
- **엔딩에서 실제 다른 배우로 교체 공개** — 같은 블로킹·대사를 ref만 바꿔 재생성.
- [제작] 친구 2명("야 너 미쳤어")은 별도 클립 분리.

### C-18. #발현 — 공황
- **축:** 공통 · **층위:** 현실 · **상태:** 프롬프트 있음 (1·2)
- 방 게임 폐인 → 엄마 "밥 먹어" → 굉음(모니터 부숨) → 무너져 욺 → "엄마 나 숨을 못 쉬겠어" → 컷 투 블랙. (발작은 방 밖 굉음으로 먼저 듣게.)
- 플래시백: 진짜+가짜 사고 섞임, 책 들고 가짜 여자 쫓음 — 이미지 누수라 지식 장부와 안 부딪힘.

### C-18b. 의사 — 정의 설명
- **축:** 공통 · **층위:** 현실 · **상태:** 미작성 · 원칙 [확정]: 충분히 설명 (concept 6장)
- 병의 이름과 정의 설명. "최근 어떤 사고를 *목격*하셨나요?" — *목격*이 미끼.

### C-19. 놀이터 재등장
- **축:** 공통 · **층위:** 환상 · **상태:** 미작성
- 부재가 길어져 관객이 먼저 풀기 직전, 수아가 현실 장소(놀이터)에 변함없이 있음. 구원 아님 — *그 자리에 있을 뿐.* 말비식 재회 충동 = 등져야 할 신호.

### C-20. 진짜 사고 — 파편 (클라이맥스 오버랩용)
- **축:** 공통 · **층위:** 현실 · **상태:** 미작성
- **메커니즘 (concept 0장):** 수아를 죽게 한 것은 지훈의 **인기척** — 옆을 지나간 움직임에 녹색불로 착각, 따라 건넘. 연출 비트: ① 달려가다 비켜 지나가며 살짝 옆모습을 돌아봄 (수아는 책 보느라 모름) ② 다 건너고 아는 얼굴이 생각나 돌아봄 ③ 바로 차가 치고 지나감.
- 완결 금지. 브레이크 소리 · 떨어지는 책 · 구르는 **빨간 운동화** · 컷. 미완의 공포.
- **톤 분리 원칙:** 횡단보도는 환상 톤(C-01, END-A — 닮게, 거울 짝)과 현실 톤(C-20 — 날것·잔혹·빠르게)으로 분리. 분리 도구는 빛/속도/소리가 구도보다 강함. 실제 구도는 생성 돌려보며 결정.
- [제작] OP-C 공통 구간 재호출 + 파편 인서트.

### C-21. #재회 — "이 책… 이제 돌려주러 갈까?"
- **축:** 공통 · **층위:** 환상 · **상태:** 미작성 (이전 세션에서 미결로 막혔던 씬 — 시간 모델 확정으로 해소)
- 책상에 수아가 그린 것 발견 → 달려나감 → 빈 자리 → "너 소식 못 들었어?" → 우체통 벤치 → 수아 기다림. "왔어?" / "미안해, 너 괜찮은 거야?" / "이제 괜찮아." → 우체통에서 책 꺼냄 → "이 책… 이제 돌려주러 갈까?"
- [진실] 회피 임무의 자진 종료 — 무너진 뒤에야 환상이 "돌려주자"고 말한다.
- + 교무실에서 수아 주소 몰래 찾는 지훈 인서트 (현실).

## 3막 — 같이 집을 찾고, 마무리

### C-22. 집 찾기 동행
- **축:** 공통 · **층위:** 환상 · **상태:** 미작성
- 수아가 길 찾기를 돕는다(자기 집이니까). 도착하는 곳 = 수아 자신의 집.
- [위장] 갈등을 딛고 화해, 같이 찾아주는 따뜻함. [진실] 회피가 한계에 닿아 가장 피하던 곳으로 떠밀림. (위치 확정은 문패/엄마가 — 장부 주의.)

### C-23. 클라이맥스 — 수아의 집 ✅핵심 대사
- **축:** 공통 · **층위:** 현실 · **상태:** 핵심 한 줄 확정, 나머지 다듬기
- 사진·빨간 운동화 발견 → C-20 파편 오버랩 → 엄마(속표지 이름 보고 손 떨림): **"장례 때도 이 책만 못 찾았어."** — 죽음 명시의 봉인이 여기서 처음 열림. 해리가 부서진다.
- 책이 엄마 손에 넘어감 = 회피 임무 종료 = 환상 존재 이유 소멸. 그 자리가 비어 있다.
- 🔲 엄마의 나머지 대사(경찰 진술 "그날 먼저 건너간 학생이 있었다" 노출 수위) — 의사 설명(개정)과 분담 조정 [OPEN].

### C-24. 병원 — 설명과 여운
- **축:** 공통 · **층위:** 현실 · **상태:** 미작성 · 원칙 [확정]: 충분히 설명, 전달 방식 [OPEN]
- "사고 전 두 사람이 알고 지낸 흔적이 없습니다." + 해리/재구성에 대한 설명 허용. 여운은 남기되 추리 숙제는 남기지 않는다.
- 병실의 우는 지훈.
- [연출 후보] 설명을 현실 원장 컷들(혼자 버전들) 위 보이스오버로 — END-B'와 결합 가능.

## 엔딩 후보 [전부 제작 → 편집에서 결정]

### END-A. 횡단보도 — "안녕, 난 지훈이라고 해" ✅
- **축:** A · **층위:** 환상(원점) · **상태:** ✅ 대사 확정
- 컷 → 횡단보도. 지훈이 이번엔 멈추고, 옆모습의 수아에게: "안녕. 갑자기 말 걸어서 미안한데 난 지훈이라고 해." → **컷 투 블랙. 반응 없음.** 인사가 허공에 매달린 채 끝.
- 닫힌 비극. 거울: C-01("맨날 그러니까") ↔ END-A("안녕") — 친밀 → 해체 → 영점.

### END-B. 인터라켄 — 벤치, 고개 돌리는 여자 [기울어짐]
- **축:** B · **층위:** 모호(판별 불가가 목적) · **상태:** 프롬프트 있음 (테스트 자산)
- 구성 미니멀 [확정]: 지훈이 언덕 벤치로 걸어가고 있다 → 벤치에 앉아 있던 여자가 고개를 돌린다 → (호수+설산). 딱 그 정도.
- 여자는 과거의 수아와 **머리(등 중간 땋은 긴 머리)도 옷(여행자 옷)도 다르다.** 벤치는 서울 벤치와 딱 보면 연상되게 닮음.
- 보조 단서 후보: "여기 오고 싶다 그랬잖아"(혼잣말) / "나 괜찮아"(엄마와 통화) — 양방향.
- ⚠️ 긴 머리는 정체 모호화 장치로만. "3년 자란 수아=생존" 단서로 쓰면 영화 토대가 흔들림.
- [제작] 서울 벤치→인터라켄 벤치 매치컷, 두 클립 따로 생성. 카메라 "날아감"은 미검증 — 어색하면 고정 와이드.

### END-B'. 투샷 → 혼자 샷 나열 [강력 — 어느 엔딩과도 결합 가능]
- **축:** 공통 · **층위:** 현실 공개 · **상태:** 미작성 (재료는 실행 쌍들: C-09R/F, C-10R/F…)
- 투샷들이 차례로 **수아만 사라진 혼자 샷**으로. 대사 없음. 시간 모델상 이것은 연출이 아니라 **문자 그대로의 진실.**
- ⚠️ 음악 결합 시: 슬픈 노래 MV처럼 되면 위로, 무음/불온한 사운드면 직면. 같은 샷, 음악이 톤을 정한다.

### END-C. 쿠키 — 빨간불 문답 토막 ✅ [엔딩 분기에 종속]
- **축:** 공통 · **층위:** 환상 · **상태:** ✅ 대사 확정 (테스트 완료)
- "빨간불엔?" / "멈춘다." / "차가 안 와도?" / "멈춘다." / "이제 잘하네." / "네가 선생님이냐?" / "응. 넌 학생."
- [진실] "이제 잘하네" = 빨간불에 멈추는 법을 배운 지훈 — 정작 수아가 죽던 날엔 멈추지 않았다. 위로처럼 들리는 가장 아픈 아이러니.

### END-D. 엔딩 크레딧 — 책 메모 + 사라지는 씬 [조건부]
- **축:** 공통 · **상태:** 미작성
- 수아 메모 + 수아 사라지는 씬 배치 = 재관람 안내서. ⚠️ END-B'와 자리 겹침 — 택일. 대사·해설 없이 이미지로만.

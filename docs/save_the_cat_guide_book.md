# PART 1: 핵심 원칙 요약

## 1장 — "무엇에 관한 영화인가?"

### 원칙
- 모든 시작은 "무엇에 관한 영화인가?"
- 이 질문에 한두 문장으로 답할 수 없으면 아직 준비 안 된 것

### 좋은 로그라인의 4요소
1. **아이러니** — 듣는 순간 감정적으로 끌리는 드라마틱한 모순
2. **장면이 그려짐** — 영화 전체가 한 문장에서 암시
3. **타깃/예산** — 누가 볼지, 얼마나 드는지 감이 잡힘
4. **죽이는 제목** — "무엇에 관한 영화인가?"에 대한 영리한 답

### 30초 피치
- 로그라인이 안 맞는 영화(Six 같은)는 30초 피치로 대체
- 피치 구조: **반복(패턴) → 패턴 깨짐("어?") → 답을 안 줌("보고 싶다")**
- EP01의 구체적 상황으로 시작하면 관객이 붙음

### 하이 콘셉트가 안 맞을 때
- 모든 영화가 하이 콘셉트일 필요 없음
- 체험적/누적적 감정이 핵심인 영화는 피치 방식을 변형
- 핵심: "관객이 왜 이걸 봐야 하는가"를 항상 생각

## 2장 — "같은 것을 줘, 단 다르게!"

### 10가지 스토리 범주

| # | 범주 | 핵심 |
|---|------|------|
| 1 | 집 안의 괴물 | 밀폐 공간 + 괴물 + 죄 |
| 2 | 황금 양털 | 길 + 동료 + 보물 (진짜 보물은 처음 찾으려던 게 아님) |
| 3 | 주전자에서 나온 지니 | 마법 능력 → 결국 그 능력 없이도 괜찮다는 걸 배움 |
| 4 | 평범한 사람에게 닥친 문제 | 평범한 사람이 음모/위기에 말려듦 |
| 5 | 통과 의례 | 성장의 관문 |
| 6 | 버디 러브 스토리 | 싫어함 → 끌림 → 사랑 → 절망의 헤어짐 → 재결합 |
| 7 | 이유를 밝히는 추리영화 | "왜?"가 엔진 |
| 8 | 바보의 승리 | 시스템이 과소평가하는 사람의 승리 |
| 9 | 집단 이야기 | 그룹 역학 |
| 10 | 수퍼히어로 | 특별한 조건을 가진 존재의 고독 |

### 사용법
- 내 영화가 어느 범주인지 알면 → 선배 작품 연구 → 그 장르의 약속 파악 → "색다르게" 지키기
- **"관객이 원하는 것을 색다른 방식으로 주라"** — 스토리텔링의 가장 오래된 원칙
- 모든 범주에 해당하면 → 샛길로 빠진 것. 하나를 골라야 함

## 3장 — "주인공이 누구인가?"

### 최상의 주인공 조건
- **가장 큰 갈등**을 겪는 사람
- **가장 긴 감정적 여정**을 끌고 갈 수 있는 사람
- **원초적 목표**를 가진 사람

### 원초적 목표 (관객이 본능적으로 공감)
- 생존
- 배고픔
- 사랑하는 사람을 지키는 것
- 죽음의 공포
- 섹스/번식

### 수식어 법칙
- 로그라인에 주인공과 악역을 **수식하는 말** 필수
- "억울하게 수감된 은행가", "늙고 잔인한 왕" — 수식어가 갈등을 내포

### 가족 관계
- 모르는 사람보다 어머니-딸, 아버지-아들, 형제, 부부가 공감에 유리

## 4장 — BS2 (블레이크 스나이더 비트 시트)

### 15개 비트 (110분 기준 페이지 번호)

```
 1. 오프닝 이미지 (1)      — 첫인상. 마지막 이미지와 대비.
 2. 주제 명시 (5)          — 영화의 테마를 암시하는 대사/장면.
 3. 설정 (1-10)            — 주인공의 세계. 뭐가 문제인지.
 4. 기폭제 (12)            — 일상이 깨지는 사건. 되돌릴 수 없음.
 5. 토론 (12-25)           — "이 길을 갈 건가?" 내적 토론.
 6. 2막 진입 (25)          — 주인공의 선택. 새로운 세계로.
 7. B스토리 (30)           — A스토리의 테마를 다른 각도로 비추는 거울.
 8. 재미와 놀이 (30-55)    — 콘셉트가 약속한 것을 보여주는 시간.
 9. 중간점 (55)            — 가짜 승리 or 가짜 패배. 전환점.
10. 악당이 다가오다 (55-75) — 모든 것이 조여옴.
11. 절망의 순간 (75)       — 모든 것을 잃은 것 같은 지점.
12. 영혼의 어두운 밤 (75-85) — 가장 조용하고 가장 아픈 순간.
13. 3막 진입 (85)          — 절망에서 벗어나 마지막 행동을 결심.
14. 피날레 (85-110)        — 해결 또는 결말.
15. 마지막 이미지 (110)    — 오프닝과 대비. 변화를 보여줌.
```

### 단편 적용 (15-20분)
- 15개 전부 독립적으로 넣지 않아도 됨
- **필수 7개**: 오프닝, 기폭제, 중간점, 절망, 3막 진입, 피날레, 마지막 이미지
- 나머지는 필수 비트 안에 흡수/압축
- 비트의 순서와 기능은 길이와 무관하게 작동

### BS2의 숨은 기능 — 분량 배분
- 페이지 번호 = 관객이 각 감정에 머무르는 시간의 비율
- "재미와 놀이"에 25분 → 행복한 시간을 충분히 보내야 나중에 잃을 때 아프다
- 어디에 씬이 몰려있는지 = 어디가 늘어지는지 (엑스레이)

### BS2는 법칙이 아니라 도구
- "이 순서대로 만들어라"가 아니라 "이 순서와 네 이야기가 어떻게 다른지 인식해라"
- 규칙을 알고 깨는 것 ≠ 모르고 안 따르는 것
- 엑스레이를 찍는 것이지 처방전을 쓰는 게 아님

## 5장 — "이제 써라"

### 체크리스트
1. ✅ 죽이는 아이디어/제목/로그라인 — 낯선 사람에게 피칭 테스트
2. ✅ 장르 숙제 — 선배 작품 연구, "같은 것을 다르게"
3. ✅ 최적의 주인공 + 악역 — 수식어, 원초적 목표
4. ✅ BS2 완료 — 40개 씬 카드 (단편은 8-10개)

### 핵심 메시지
- 준비는 끝이 없다. 어느 시점에서는 뛰어들어야 한다.
- "심해로 내려가라" — 불완전한 상태로 시작하는 것이 정상

## 6장 — 시나리오 불변의 법칙들

### 고양이를 구하라 ⭐
- 주인공이 초반에 관객의 호감을 사는 **작은 행동**
- 세계를 구하는 게 아니라 "이 사람은 기본적으로 괜찮다"는 신호
- 어설프고 약한 캐릭터는 **어설픔 자체가 호감** — "이 사람이 괜찮아졌으면 좋겠다"

### 풀장 안의 교황
- **설명(exposition)을 재미있는 상황 안에 숨기는 기법**
- 관객이 "뭐야 저건?"하고 쳐다보는 사이에 정보 전달
- 설명이 필요하면 최소한 재미있게

### 중복되는 마법
- **같은 영화에서 같은 종류의 초자연적 장치를 두 번 쓰지 마라**
- 마법 예산은 한 번에 다 써야 함
- The Beach 적용: 루프가 유일한 "마법". 바텐더나 xx역 사람에게 별도의 마법을 붙이면 안 됨

### 취재진은 빼라
- 설명을 위한 캐릭터(기자, 뉴스 등)를 넣으면 게으른 시나리오
- 보여줘야 할 걸 텍스트/대사로 때우지 마라

### 흑인 수의사
- 지나치게 특이한 조합은 관객의 주의를 분산시킴
- 캐릭터의 특이함이 스토리에 기여하지 않으면 빼라

## 7장 — "어디가 고장 난 걸까?" 테스트

### 9개 체크리스트

```
1. 주인공이 능동적인가?
   → 스토리의 모든 단계에서 자기 욕망/목표로 움직이는가?

2. 대사로 플롯을 설명하지 않는가?
   → 행동으로 보여주는 대신 입으로 설명하고 있지 않은가?

3. 악역이 충분히 강한가?
   → 주인공에게 적절한 도전을 하는가? 둘 다 이 영화에 어울리는가?

4. 중간점 이후 점점 빨라지는가?
   → 3막으로 갈수록 더 많은 것이 드러나는가?

5. 감정적으로 단조롭지 않은가?
   → 코미디, 드라마, 슬픔, 좌절이 적절히 섞여있는가?

6. 대사가 단조로운가?
   → 말투만으로 캐릭터를 구별할 수 있는가?

7. 조연이 구별되는가?
   → 외양, 말투, 행동방식이 각각 독특한가?

8. 주인공의 시작점이 충분히 낮은가?
   → 성장의 여정이 충분히 긴가?

9. 원초적인 이야기인가?
   → 사랑, 생존, 가족, 복수 — 본능적 욕구를 향하고 있는가?
```

### 핵심 태도
- **"의심이 들면 고쳐라."** "아무도 눈치 못 채겠지"는 거짓말.
- 지망생은 "형편없어!" 하고 포기. **프로는 "어떻게 고칠지 알겠어!" 하고 고침.**
- 퍼즐을 되풀이해서 맞추는 것. 연습할수록 빨라짐.

## 연습에서 발견된 패턴

Save the Cat 1장 연습(장르 전환, 학교, 대결, 살인마)을 통해 발견:

### 핵심 DNA
1. **정체의 이중성** — 가짜 신분, 가짜 관계, 기억 교환
2. **가짜가 진짜가 되는 순간** — 위장이 진심이 되는 긴장
3. **"나는 누구인가"** — 정체성의 혼란과 발견

### 선호하는 감정
- 알면서도 선택하는 인물
- 엔딩이 전체를 재맥락화하는 구조
- **관객이 간절히 바라지만 영화가 허락하지 않는 결말**
- 비극 없는 악당 — 모든 캐릭터에 이해 가능한 동기

### Six와 The Beach의 공통점
- "닿을 수 없는 사람"
- "다른 존재로 살아야 하는 사람"
- 반복 속의 유일한 변수
- 관객이 아는 것을 인물은 모르는 구조 (히치콕의 서스펜스)

# PART 3: 실전 체크리스트

## 시나리오 쓰기 전 체크

```
□ 로그라인/피치가 한두 문장으로 되는가?
□ 낯선 사람에게 피칭했을 때 "보고 싶다"가 나오는가?
□ 어떤 장르(10가지 범주)인지 아는가?
□ 같은 범주의 선배 작품을 3편 이상 봤는가?
□ 주인공의 수식어가 있는가? ("진실을 알면서 침묵한 남자")
□ 악역의 수식어가 있는가?
□ 원초적 목표가 있는가? (생존, 사랑, 보호, 복수)
□ BS2 비트가 잡혔는가?
□ 오프닝과 마지막 이미지가 대비되는가?
```

## 시나리오 완성 후 체크 ("어디가 고장 난 걸까?")

```
□ 주인공이 능동적인가?
□ 대사로 플롯을 설명하지 않는가?
□ 악역이 충분히 강한가?
□ 중간점 이후 점점 빨라지는가?
□ 감정적으로 단조롭지 않은가?
□ 대사가 단조로운가? (인물별 말투 구별 가능?)
□ 조연이 구별되는가?
□ 주인공의 시작점이 충분히 낮은가?
□ 원초적인 이야기인가?
```

## 장면 쓸 때 체크

```
□ 이 장면에 고양이를 구하는 순간이 있는가? (특히 1막)
□ 설명이 "풀장 안의 교황"으로 숨겨져 있는가?
□ 중복되는 마법은 없는가?
□ 취재진(설명용 캐릭터)이 없는가?
□ 이 장면은 BS2의 어떤 비트인가?
□ 이 비트에 씬이 너무 몰려있지 않은가? (분량 배분)
```
